Parenthesis ()

Parenthesis can be used in Javascript for a few different thing. They are used to call functions to execute when a function is defined. They can also be used to group conditions or statements.

Brackets []

Brackets are most commonly used for arrays. They are used to both define arrays and access data in arrays.

Braces {}

Also known as curly brackets (cute). These are used to group sets of code together. Most commonly used to group together statments which come under a function.

Single '' and Double "" Quotes

Both single and double quotes can be used to define a string. Single quotes can be somewhat tricky if your string inside it contains an apostrophy (which you'd then have to define as part of the string and not the closure of the string using a forward slash /.
