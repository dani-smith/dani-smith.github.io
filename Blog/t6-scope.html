What is 'Scope' in Javascript

Javascript has two scopes, global and local. Scope refers to the accessibility of variables declared in Javascript. Variables declared outside of functions are known as global scopes, which can be used anywhere in the Javascript code. Variables declared inside functions are known as local scopes, meaning they are local to the function they are declared in and can only be used inside of the function the are declared in.
